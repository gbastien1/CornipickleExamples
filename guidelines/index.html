CORNIPICKLE GUIDELINES

CORNIPICKLE TESTS

GNOME'S HIG
  • BUTTONS
    • Two close buttons have the same width
    • Affirmative and destructive buttons
    • Button labels should use header capitalization
    • No right click or double click should be assigned to a button
  • CHECKBOXES
    • Sentence capitalized checkbox labels
    • Checkbox groups should be vertically aligned
    • Related checkboxes should be aside or one above the other(not working)
    • Provide an accesskey on all checkbox labels
  • MENUS
    • Menu items should use Heading Capitalization
    • Provide an accesskey for each menu item
  • POPOVERS
    • Popover's size lesser than 1/3 of container
    • No popover should appear without user action
  • SPIN BOXES
    • Spin box guideline 1
  • RADIO BUTTONS
    • At least two radio buttons
    • Use sentence capitalization for labels
    • Align radio button groups vertically
    • Select only one radio button at a time
  • TEXT FIELDS
    • Text field's width fits the content
  • SLIDERS
    • Give real time feedback of moving slider
  • TABS
    • Use at most one row of tabs
