The guideline to respect:

Use sentence capitalization for checkbox labels, for example "Use custom font".

Link to the guideline (3rd bullet point)

Detect it with Cornipickle

# We define a predicate using the construct # "We say that <arguments> is/are <predicate name> when". # requirement: the labels must be placed inside <label></label> tags # and they must contain letters only We say that $x is capitalized when ( $x's text matches "^([A-Z][a-z]*(?:\s[a-z]*)*)$" ). """ @name Sentence capitalized labels @description All checkbox labels should be sentence capitalized @severity Warning """ For each $label in $(label) ( $label is capitalized ).

Procedural Equivalent

var t = new RegExp("^([A-Z][a-z]*(?:\s[a-z]*)*)$"); $("label").each(function() { if (t.match($(this).text) === null) console.log("Error"); });
