The guideline to respect

Do not use too many tabs. If you cannot see all the tabs without scrolling or splitting them into multiple rows, you are probably using too many and should use a list control instead.

Detect it with Cornipickle

We say that $tab does not overflow when ( There exists $container in $(.tab_container) such that ( ($tab's right is lesser than $container's right) Or ($tab's right equals $container's right) ) ).
